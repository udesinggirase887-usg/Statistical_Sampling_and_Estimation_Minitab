# 📊 Statistical Sampling and Estimation using Minitab

This repository contains practical implementations of statistical concepts using **Minitab** as part of M.Sc. Statistics coursework. It covers a wide range of topics including sampling techniques, estimation methods, regression analysis, and inequality measurement.

---

## 📁 Project Overview

The project is based on practical assignments from statistical topics such as:

- Random Sampling Techniques  
- Sampling Distributions  
- Estimation of Population Parameters  
- Sample Size Determination  
- Stratified Sampling  
- PPS (Probability Proportional to Size) Sampling  
- Ratio and Regression Estimation  
- Cluster Sampling  
- Lorenz Curve and Gini Coefficient  

Each topic includes datasets, Minitab analysis, and interpretation of results.

- **Minitab** – for statistical analysis
